# ArchiCalc Guide GitHub Pages

Folder ini siap dipakai untuk GitHub Pages.

## Isi

- `index.html` - landing page aplikasi
- `privacy-policy.html` - privacy policy untuk AdMob
- `app-ads.txt` - publisher declaration untuk AdMob
- `downloads/ArchiCalc-Guide-Bilik-Arsitektur.apk` - APK aplikasi
- `logo.png` - logo/brand image

## Cara Upload

1. Buat repository baru di GitHub, misalnya `archicalc-guide`.
2. Upload semua isi folder ini ke root repository.
3. Buka repository `Settings` > `Pages`.
4. Pada `Build and deployment`, pilih `Deploy from a branch`.
5. Pilih branch `main` dan folder `/root`.
6. Simpan, lalu tunggu URL GitHub Pages aktif.

Contoh URL:

```txt
https://username.github.io/archicalc-guide/
https://username.github.io/archicalc-guide/privacy-policy.html
https://username.github.io/archicalc-guide/app-ads.txt
```

Untuk AdMob jangka panjang, custom domain lebih disarankan.
